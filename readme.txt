Данная программа является аналогом ImCurr от Ingenious,
исходный код которой доступен по ссылке:

https://github.com/IngeniousA/ImCurr

Отличия:

* imCurrJVM разработана на языке Java, что позволяет запускать
её на любой ОС, поддерживающей Java Virtual Machine (JVM).

* imCurrJVM обладает простым графическим интерфейсом.

Принцип работы:

Любой файл можно открыть с помощью текстового редактора, доба-
вить несколько произвольных строк, но при этом файл не будет
повреждён. Таким образом, можно прятать текст от посторонних
глаз. На этом принципе и основана работа ImCurr и imCurrJVM.

Системные требования:

Минимальные:

* Java 8.0
* 128 MB RAM

Настройка программы:

В данный момент для настройки доступны только языки. Языковой
файл находится в директории с исполняемым файлом и назвается 
lang.txt. Вы можете редактировать его, используя встроенный 
интерфейс (нажмите на кнопку Settings). Таким образом, вы можете
создать перевод на любой язык. Если вам вдруг стало нечего делать 
и вы действительно написали перевод - откройте Issue.

Так же можете скачать официальный перевод на русский язык: lang_ru.txt.
После загрузки скопируйте его в директорию с исполняемым файлом, удалите
файл lang.txt и переименуйте lang_ru.txt в lang.txt.
